Google Fusion Table Wrapper API for PHP

*** Version 2.0 ***

- Upgraded to use Google Fusion Tables API V1 (prior version was for GFT API Beta).

*** Instructions and Notes ***

- Include the FusionTableWrapper.php in your PHP script file. For example:
  include_once "FusionTableWrapper.php";
  
- Make sure the FusionTableJSONParser.php and the FusionWebRequest.php files are
  included in the same folder as the FusionTableWrapper.php file.
  
*** Please refer to the FusionTableWrapperTest.php file for examples ***
